## 演算子
配列に対して以下の演算子を直接利用することが利用できます。演算は配列の各要素に対して作用し、演算結果は配列で返ります。

  * **単項演算子:**

| 演算子 | 名前    | 補足                         |
|:----- |:------ |:-----------------------------|
| `+`   | 単項+   | 特に何もしない                 |
| `-`   | 単項-   | 正負を反転                    |

  * **二項演算子:**

| 演算子 | 名前   | 補足                         |
|:----- |:------ |:----------------------------|
| `+`   | 加算   |                             |
| `-`   | 減算   |                             |
| `*`   | 乗算   | 内積・外積ではない             |
| `/`   | 除算   |                             |
| `\`   | 除算   | `x \ y` は `y / x` と等価    |
| `^`   | べき乗 |  `x ^ y` は `x` の `y` 乗    |

  * **比較演算子:**

| 演算子 | 名前   | 意味                        |
|:----- |:------ |:---------------------------|
| `==`  | 等価   |                            |
| `!=`  | 非等価 |                            |
